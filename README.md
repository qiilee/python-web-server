# python-web-server
> Python3 实现简单的web服务器

## 项目运行（python3.6）
``` bash
# 克隆到本地
git clone https://github.com/qiilee/python-web-server.git

# 进入文件夹
cd python-web-server

# 开启本地服务器
python3 web.py

# 浏览器访问
http://127.0.0.1:8030/index.html
```
